# MySQL最终回归清单 v1

本文档用于在 `DB_CLIENT=mysql` 模式下，对 TA 选课系统进行上线前最终回归验证。

建议启动方式：

```bash
cd /Users/yanren/Documents/Playground
DB_CLIENT=mysql HOST=0.0.0.0 ./node-v22.22.1-darwin-arm64/bin/node server.js
```

建议测试原则：

- 优先验证主流程，不先测边缘功能
- 每项记录结果：`通过 / 不通过 / 待确认`
- 若发现问题，记录：
  - 页面
  - 角色
  - 操作步骤
  - 实际结果
  - 预期结果
